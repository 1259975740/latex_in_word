LaTeX in Word is a GPL-licensed tool that allows equations to be used in
Microsoft Word documents.  The client-side of the program is implemented as VBA
macros in the document "LaTeXinWord.doc" along with instructions.  Hence, this
file contains the source code, implementation, and documentation.

Complete license information can be found in the file
"GPL_with_LinW_v_0_3_0.txt".  Updates and additional information can be found
on the SourceForge project page:

http://sourceforge.net/projects/latexinword
